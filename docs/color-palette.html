EchoCards Color Palette

A comprehensive guide to colors used throughout the EchoCards application. For development team use only.

Theme Colors (Base)

These are the base theme colors that define the overall look and feel of the application.

Light Background
Background #ffffff
Used for the main background of the application in light mode.
bg-background (light)
Tailwind: bg-white
Dark Background
Background (Dark) #111827
Used for the main background of the application in dark mode.
bg-background (dark)
Tailwind: dark:bg-gray-900
Primary
Primary #3b82f6
Used for primary buttons, links, and interactive elements. Our main brand color.
text-primary, bg-primary
Tailwind: text-blue-500, bg-blue-500
Primary (Dark)
Primary (Dark) #3b82f6
Used for primary buttons, links, and interactive elements in dark mode.
text-primary, bg-primary
Tailwind: dark:text-blue-500, dark:bg-blue-500
Muted
Muted #f1f5f9
Used for subtle backgrounds, disabled states, and secondary UI elements.
bg-muted
Tailwind: bg-slate-100
Muted (Dark)
Muted (Dark) #1e2937
Used for subtle backgrounds, disabled states, and secondary UI elements in dark mode.
bg-muted
Tailwind: dark:bg-slate-800
Destructive
Destructive #ef4444
Used for error states, delete actions, and to indicate warnings.
bg-destructive
Tailwind: bg-red-500
